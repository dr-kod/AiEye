# AI-Eye 

This project introduces a **wearable AI-driven assistant** designed to help **blind and visually impaired users** perceive and interact with their environment through **multi-camera vision** and **real-time audio guidance**.

## Overview
The system uses **multiple cameras** and **artificial intelligence** to analyze the surrounding environment from all directions, detecting objects, obstacles, people, and relevant events. It translates visual information into **natural, spoken feedback** using embedded speakers.
 
## Key Features
- Lightweight wearable design, similar to glasses or a head-mounted frame
- Multi-camera coverage for full environmental awareness
- AI-powered object, obstacle, person, and text recognition
- Real-time 3D spatial awareness with directional audio
- Natural voice feedback for navigation and safety
- Shock-resistant and durable construction
- Software updatable and extendable

## Purpose
The primary goal is to increase **independence, safety, and quality of life** for blind individuals by providing them with a better understanding of their surroundings and enabling confident mobility.

## Technical Concept
The device integrates:
- Multiple HD cameras for multi-angle vision (front, back, sides)
- Onboard processing powered by AI and computer vision algorithms
- OCR (Optical Character Recognition) for text reading
- 3D audio module for spatial sound feedback
- Compact battery and lightweight frame design

## Future Developments
- Integration with smartphones for cloud-based updates and enhanced recognition accuracy
- Voice command interaction and personal assistant features
- GPS-based navigation and route guidance
